mac Command Line Tools问题

2019-01-22 Tue
Mac
123 words 1 min read times read

Contents

更新MacOS后，Command Line Tools 出问题。

  • 原因：就是每次MacOS更新都会自动卸载xcode。
  • 解决方案：在terminal运行 xcode-select –install。就是手动重装xcode，然后用xcode-select --install来安装一些xcode插件。
