一些笔记

screen翻页

使用screen让程序在后台运行比较方便，但是查看程序输出时，需要用到下面的快捷键。 * [激活翻页]Ctrl+A+[ * [上下翻页]Ctrl-u,Ctrl+d

ssh-add key

使用ssh-add key之后，再次使用ssh连接服务器，就不用再输入密码了。

根据宿主机的pid查找容器的id

查找容器的id

1

docker inspect -f "{{.Id}}"  $(docker ps -q) |grep <PID>

查找k8s pod name

1

docker inspect -f "{{.Id}} {{.State.Pid}} {{.Config.Hostname}}"  $(docker ps -q) |grep <PID>
